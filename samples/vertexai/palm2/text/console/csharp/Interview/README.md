# VertexAI Palm2 text, console, C#

A VertexAI Palm2 text sample to run on console with C#.

First, change `PROJECT_ID` in [config.sh](../../config.sh) to your own.

Run [setup.sh](../../setup.sh) to enable required services and setup
authentication:

```bash
../../setup.sh
```

Build:

```bash
dotnet build
```

Run:

```bash
dotnet run
```
